CMRX Linux-based window server demo
===================================

<p align="center">
  <img alt="Window server screenshot" src="doc/cmrx-linux-window-server.png">
</p>

This is a simple demo of a virtualized framebuffer device, windowing server and a bunch of small applications using this infrastructure. The point is to demonstrate capabilities of CMRX when running on Linux and to stress the CMRX kernel
